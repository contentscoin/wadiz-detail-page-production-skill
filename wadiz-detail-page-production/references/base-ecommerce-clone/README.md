# Base Ecommerce Clone

This folder is a local clone of the older `ecommerce-detail-page` skill's core
references and helper scripts. It exists so the Wadiz production skill can remain
self-contained when shared or installed without the older skill.

Canonical upstream source: https://github.com/aisyncclub/detail_page_codex_skill
OpenCrab platform source for Wadiz pack retrieval: https://opencrab.sh

Prefer `../base-ecommerce-pipeline.md` for the clean operating contract. Use
these cloned files when exact legacy behavior, gallery scripts, or detailed
reference tables are needed.
