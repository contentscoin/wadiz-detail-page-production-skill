# 샘플 상품 Brief

이 샘플은 스킬 사용 형식을 보여주기 위한 가상의 입력입니다. 실제 제작 시에는 공식 URL, 판매자 제공 자료, 가격표, 배송/환불 정책을 기반으로 fact map을 다시 만들어야 합니다.

## 사용자 요청 예시

```text
$wadiz-detail-page-production을 사용해서 아래 상품의 와디즈 스타일 상세페이지 컷 구조와 제작 계획을 만들어줘.

상품명: 커스텀 골프공 정기구독 서비스
카테고리: 골프 / 스포츠용품 / 정기구독
목표 채널: 스마트스토어 상세페이지
타깃: 라운드를 자주 나가고 골프공을 자주 잃어버리는 30-50대 아마추어 골퍼

핵심 특징:
- 매월 골프공을 정기 배송
- 사용 주기별 옵션 선택 가능
- 골프공에 이름, 로고, 문구 인쇄 가능
- 선물용 패키지 옵션

제공 자료:
- 상품 사진 3장
- 판매자가 확인한 가격표
- 배송 정책
- 환불/교환 정책

주의:
- 비거리 향상 보장 표현 금지
- 국내 최초, 업계 1위, 무조건 보상 같은 claim 금지
- 기존 이미지를 그대로 합성하지 말고 제품 디테일 분석 후 전체 장면 생성 방향으로 기획
```
